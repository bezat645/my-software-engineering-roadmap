# Month 01: Full-Stack Web Foundations & Architecture

A structured, production-grade learning repository tracking my 14-month technical roadmap from web foundations to advanced full-stack and cloud infrastructure engineering.

## Week 01: Semantic HTML5 & Version Control Architecture

This week's milestones focused heavily on establishing a clean document object model (DOM), understanding content hierarchies, accessible form validation, and professional local/remote Git version control workflows.

###  Core Milestones Completed

* **`index.html` (Semantic Portfolio Structure):** Implemented clean HTML5 structural elements (`<header>`, `<main>`, `<section>`, `<article>`) and crisp internal bookmark navigation.
* **`form.html` (Accessible Data Gateway):** Engineered an accessible input form utilizing standard semantic grouping layout controls (`<fieldset>`, `<legend>`) and strict explicit label pairing (`<label for="...">`).
* **`resume.html` (Project Milestone - Structural Resume):** Synthesized a complete capstone resume layout combining advanced content wrapping, layout block definitions, and native HTML5 constraint validation parameters.

###  Technical Stacking & Tools
* **Languages:** HTML5 (Semantic specifications, text level controls, character entities)
* **Version Control:** Git (Conventional Commits specification), GitHub
